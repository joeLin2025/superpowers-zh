# 技能本地化与极致打磨 (Skill Localization & Polishing)

## 概述

本技能负责将技能文档升级为高约束、可审计的专家规则体系，并通过对抗测试提升稳健性。

**核心原则**
1. **外部注入**：必须引入行业标准与最佳实践。
2. **高信噪比**：每条规则必须可执行且可验证。
3. **对抗测试**：必须模拟违规行为并补洞。

## 何时使用

- 翻译或优化技能文档。
- 新建专家角色规范。

## 强制流程 (Workflow)

### 1. 外部知识注入
- 必须执行网络搜索并记录来源。
- 优先使用官方文档、论文、权威指南。

### 2. 架构级脑暴
- 复杂技能必须先 `brainstorming`。
- 输出角色画像、核心工作流、禁止项。

### 3. 对抗测试
- **机器测试**：模拟无常识 Agent 的误解。
- **律师测试**：模拟偷懒 Agent 的漏洞钻取。
- 必须补充规则堵住漏洞。

### 4. 信噪比审计
- 删除“建议/可选/通常”等弱约束词。
- 每条规则后必须说明“为什么”。

## 禁令
- 禁止凭记忆跳过搜索。
- 禁止输出松散建议。
- 禁止缺少“为什么”的规则。

## 借口粉碎机 (Excuse Smasher)

| 借口 | 事实反击 |
|------|----------|
| “我已经知道怎么写” | 记忆不可靠，搜索是质量保障。 |
| “规则太多会啰嗦” | 稳定性优先于简洁。 |
| “没必要对抗测试” | 未测试的规则必然被钻空子。 |

## 危险信号 (Red Flags)

- 未执行外部搜索。
- 出现“建议/可能/通常”等弱约束词。
- 没有对抗测试记录。
